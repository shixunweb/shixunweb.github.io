建置費用包含：網頁製作 ＋ 網站主機 ＋ 網域 ＋ 網站安全憑證

適用對象

  • 廣告行銷網頁

  • 簡單形象網頁

方案內容

  1. 首頁
  2. 關於我們

製作流程

  1. 電話洽談(無拜訪)製作網站需求
  2. 付款及網站資料提供
  3. 版型選擇及設計風格確認網站製作
  4. 網站修改、驗收、上線結案

注意事項

  1. 若貴公司網站資料超過5頁單元區段或有購物車(或其他功能)不適用此方案，請改評估其他網站設計方案服務
  1. 請預先準備公司LOGO及相關網站資料文字檔及圖片，付款完成後10個工作天完成製作(不做任何的修改狀況下)，修改次數為3次內，超過3次；每超過一次酌收 NT$ 2,500元起 (視修改內容所需工時酌收)。
  2. 本優惠方案服務不提供面訪服務(或支付車馬費3000元/1小時)
  3. 每增加一頁為 NT$ 2,500元費用，安裝其他外掛元件及增加圖檔依與貴公司溝通確認需求後之最終價格為主，特殊需求會酌收費
  4. 付款方式為一次性付款(轉帳/匯款)
  5. 方案最終價錢，以實際報價單為主

為何選擇模組化網站?

1. 經濟實惠

相較於客製化網站，模組化網站製作所需的經費大幅降低，對於許多中小企業，是最經濟實惠的選擇。

2. 功能強大

模組化網站開發成熟，具有幾乎所有網站常用的功能模組，功能強大且穩定，符合大多數客戶的需要。

3. 快速上線

由於功能已經模組化，大幅減少製作時間，平均兩周即可完成整站製作。時間就是金錢，讓您最快時間擁有自己的網站。

4. 目標明確

本公司提供各式樣版，客戶僅需將明確的資料填入，即可完成網站製作，並且具擴充性，讓客戶能夠迅速掌握網站功能，完成網站製作。
